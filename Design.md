Purpose:
 Build a modern nonprofit website for Core Immersive, positioned as an organization focused on Creative Technology for Social Impact.
The site must communicate that Core Immersive is not a traditional technology education program. It is a movement helping people use emerging technologies to express ideas, solve problems, and shape the future.
Design tone:
 Inspirational, cinematic, modern, mission-driven.
Visual style should feel like:
creative studio
 innovation lab
 nonprofit movement
NOT like:
coding bootcamp
 online course platform
 corporate training program
Use large visuals and immersive storytelling.
Leave placeholders for images and video content that will later be generated using Sora.

Site Architecture
Main pages:
Homepage
 Programs
 Creator Lab (Flagship Program)

 Stories (Student Journey)
 About Core Immersive
 Get Involved
Navigation should be simple and modern.
